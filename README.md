# 🤖 LLM-Powered Intelligent Predictive HR System (People Analytics)

An end-to-end **AI-powered HR Analytics Dashboard** built using **Streamlit, Machine Learning, and Gemini LLM** to help HR teams and leadership analyze **employee attrition risk, employee-level HR insights, resume screening, and decision support** from one unified dashboard.

This project combines **predictive HR analytics** with **LLM-based reporting and chatbot assistance** to create a more intelligent and interactive people analytics system.

---

# 📌 Project Overview

The goal of this project is to build a **smart HR decision-support platform** that can help organizations with:

- 📉 **Employee attrition risk analysis**
- 🏢 **Department-level workforce insights**
- 🚨 **High-risk employee identification**
- 📝 **AI-generated employee risk reports**
- 📄 **Resume screening and candidate-job fit analysis**
- 🤖 **HR Copilot chatbot for analytics interpretation and HR recommendations**

The system is designed as a **multi-page Streamlit dashboard** with a premium UI and multiple business-facing modules.

---

# ✨ Key Features

## 1️⃣ Executive Summary
A leadership-focused overview of the HR analytics system showing:
- high-risk employee count
- average attrition risk
- top risk department
- overtime exposure
- executive interpretation and recommended actions

## 2️⃣ Attrition Dashboard
Interactive analytics dashboard for workforce attrition insights:
- department-wise employee analysis
- job role distribution
- overtime and attrition visualizations
- KPI cards for attrition rate and employee metrics
- high-risk employee table with download support

## 3️⃣ Employee Reports
AI-generated employee-level risk reporting system:
- employee report selection
- employee snapshot cards
- risk context and metadata
- downloadable employee HR risk report

## 4️⃣ Resume Screening & Candidate Evaluation
LLM-assisted resume screening workflow:
- upload resume (PDF/TXT)
- compare against job description
- skill match / experience match / education match scoring
- candidate fit gauge chart
- AI recruiter-style candidate report generation

## 5️⃣ HR Copilot Chatbot
Gemini-powered HR assistant for dashboard insights:
- attrition-related Q&A
- department-level risk interpretation
- candidate evaluation summary
- HR action recommendations
- dashboard decision-support assistant

## 6️⃣ Project Statistics
Technical and operational overview page showing:
- dataset scale
- generated reports count
- candidate report status
- module coverage
- project health overview

---

# ❓ Problem Statement

Traditional HR dashboards often focus only on descriptive reporting and do not provide enough intelligence for proactive decision-making.  
This project addresses that gap by combining:

- 📊 **predictive analytics** for attrition risk
- 🧠 **LLM-generated employee and recruiter reports**
- 💬 **interactive decision support through a chatbot**
- 📄 **resume screening automation**

The result is a more complete **People Analytics platform** that supports both **employee retention** and **candidate evaluation** workflows.
